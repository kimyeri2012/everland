** 실습 준비 **
1. c:\xampp\xampp-controll.exe 실행
2. 윈도우 사용자 + My-SQL 설치된 경우
- Ctrl+Alt+Del -> 작업 관리자 -> mysqld 종료
3. Apache, My-SQL 항목 start 클릭


4. vscode - 폴더 열기 - c:\xampp\htdocs
5. c:\xampp\htdocs안에 web_project 폴더 선택
   폴더 없는 경우 별도 생성
6. 지난 내용 열기
   자료 필요한 경우 "공유" 폴더 > web_project 폴더 복사하여
   c:\xampp\htdocs안에 붙여넣기
